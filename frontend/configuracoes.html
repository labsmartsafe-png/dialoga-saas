⚙️ Configurações de WhatsApp

Conecte seu número via Meta Cloud API e envie uma mensagem de teste.

🔗 Conectar via Meta API

Pegue o Token de acesso e o Phone Number ID no painel da Meta (Business → WhatsApp → Configuração da API).

O token é criptografado antes de salvar e nunca é exibido de volta.

📤 Enviar mensagem de teste

Escolha uma conexão e envie um texto. Em modo de teste da Meta, o destino precisa estar na lista de números permitidos, e a 1ª mensagem deve ser dentro da janela de 24h.

📱 Conectar via QR Code (não-oficial)

Conecte um número escaneando um QR Code, sem conta na Meta. Atenção: é um método não-oficial — há risco de banimento do número pelo WhatsApp, especialmente com volume alto. Use um número dedicado e não o seu principal.
